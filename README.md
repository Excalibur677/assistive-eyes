



\# Assistive Eyes — Medicine Identification System



A dual-mode medicine label reader built for both sighted users

and visually impaired individuals.



\## Features

\- Live webcam medicine label scanning

\- OpenCV image preprocessing (glare \& geometry correction)

\- EasyOCR text extraction

\- FuzzyWuzzy medicine name matching

\- Voice interface for blind users (Web Speech API)

\- Admin dashboard for database management



\## Tech Stack

\- Backend: Django 6.0 + Python 3.13

\- Vision: OpenCV + EasyOCR

\- Matching: FuzzyWuzzy

\- Database: SQLite

\- Frontend: HTML, CSS, JavaScript
